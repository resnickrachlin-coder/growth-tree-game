# 百知思维模型项目 - 长期记忆

## 项目概述
- 整理《百知思维模型》书籍 101 个思维模型，按方案B正版1-101顺序
- 每个模型以 JS 代码块格式存储，包含 modelId, modelName, origin, coreConclusion, logicLayer, sceneCase, commonMistake, operationStep, thinkQuestion, gridCombine, selfCheck, sevenDayPlan, remind 等字段
- 文件存放在: 百知思维模型/ 目录下，按部分分 7 个文件

## 注意事项
- 用户发送的模型编号可能与目录对不上，需按模型名称匹配到正确编号位置
- 格式统一用 ```js ``` 代码块包裹
- 部分模型的 part 字段与实际目录归属可能不一致，以用户发送的模型名称匹配目录编号写入
